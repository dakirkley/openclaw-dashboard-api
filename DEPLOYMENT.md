# Deployment Info

## API Server

This is the backend API that bridges OpenClaw instances with the dashboard.

## Local Development
```bash
npm install
npm start
```

Server runs on port 3001 by default.

## Environment Variables
- `PORT` - Server port (default: 3001)
- Database connection vars as needed
